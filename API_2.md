  </li>
  <li>
    <a href="https://docs.langchain4j.dev/">LangChain4j</a>: É uma biblioteca em Java projetada para simplificar a integração de aplicações com Grandes Modelos de Linguagem (LLMs), facilitando a criação de sistemas baseados em Inteligência Artificial.
  </li>
  <li>
    <a href="https://ollama.com/">Ollama</a>: É uma ferramenta que permite baixar e executar modelos de linguagem de código aberto (LLMs) diretamente na sua máquina local de forma muito simplificada e eficiente.
  </li>
  <li>
    <a href="https://lmstudio.ai/">LM Studio</a>: É um aplicativo desktop com interface amigável que permite descobrir, baixar e rodar modelos de linguagem (LLMs) locais e offline, facilitando testes e experimentações com diferentes IAs.
  </li>
</ul>

<h2 style="font-family:roboto;"> Contribuições Individuais :dart:</h2>
  
  <h3> Atribuições como Desenvolvedor Back-end</h3>
  <p align="justify" style="font-family:roboto;"> Atuei como desenvolvedor neste projeto focando na infraestrutura de back-end e na integração de Inteligência Artificial. Fui responsável por estruturar a conexão com o banco de dados e construir a ponte de comunicação entre a aplicação e o modelo de linguagem (LLM). Além do desenvolvimento, também liderei a fase de testes e validação técnica, avaliando diferentes serviços e executores de modelos para definir a arquitetura de IA mais eficiente para a nossa solução.

Fui responsável por realizar a configuração do acesso ao banco de dados. Além de estabelecer a infraestrutura, conduzi testes de conexão para validar a comunicação, garantindo que a integração fosse concluída com sucesso e operasse com estabilidade.

  <h2 style="font-family:roboto;"> Aprendizados Efetivos :book:</h2> 

  Durante o desenvolvimento desse projeto de chatbot para a fatec, pude adquirir diversos aprendizados significativos que contribuíram para o meu crescimento e desenvolvimento como desenvolvedor.

Este projeto foi uma excelente oportunidade para aprofundar meus conhecimentos práticos em Back-end Java e Inteligência Artificial. Pude solidificar minha base na arquitetura de aplicações, entendendo a fundo a organização limpa das classes e o fluxo de conexão com o banco de dados. O grande diferencial, no entanto, foi mergulhar no ecossistema das IAs de execução local: aprendi na prática como dimensionar, configurar e rodar pequenos modelos de linguagem (LLMs) diretamente na máquina, integrando-os perfeitamente ao sistema.
