  <p align="justify" style="font-family:roboto;"> O projeto tem como finalidade fornecer uma plataforma web que permita aos usuários acessar, visualizar e compreender de maneira eficaz todos os processos e artefatos da metodologia ágil Scrum. A interface será projetada para oferecer uma experiência de navegação intuitiva e de fácil acesso.
Uma das principais características do site é a facilidade de navegação, aliada a um conteúdo organizado para simplificar o aprendizado do Scrum. Além disso, o portal conta com questionários ao final de cada módulo para validar o conhecimento do usuário

<ul>
  <li>
    <a href="https://www.mysql.com/">MySQL</a>: É um sistema de gerenciamento de banco de dados (SGBD) relacional, ou seja, que utiliza a linguagem SQL como interface. Lançado sob a licença GPL, possui como desenvolvedora a Oracle Corporation.
  </li>
  <li>
    <a href="https://html.spec.whatwg.org/multipage/">HTML</a>: É uma linguagem de marcação utilizada para estruturar e organizar o conteúdo das páginas da web. Com suas tags e elementos, permite a criação de textos, imagens, links e formulários, fornecendo a base fundamental para a construção de sites.
  </li>
  <li>
    <a href="https://www.w3.org/Style/CSS/Overview.en.html">CSS</a>: É uma linguagem de estilo utilizada para definir a apresentação e o layout visual das páginas da web.
  </li>
  <li>
    <a href="https://www.python.org/">Python</a>: É uma linguagem de programação de alto nível, versátil e de propósito geral. Possui uma sintaxe clara e legível, sendo amplamente utilizada em desenvolvimento web back-end, ciência de dados, automação e inteligência artificial.
  </li>
  <li>
    <a href="https://www.figma.com/">Figma</a>: É uma ferramenta de design de interface (UI) e experiência do usuário (UX) baseada em nuvem. Permite a criação de protótipos, wireframes e a colaboração em tempo real entre equipes de design e desenvolvimento.
  </li>
  <li>
    <a href="https://trello.com/">Trello</a>: É uma ferramenta de gerenciamento de projetos baseada em quadros virtuais (metodologia Kanban) que permite organizar tarefas e colaborar em equipes de forma visual e intuitiva.
  </li>
  <li>
    <a href="https://flask.palletsprojects.com/">Flask</a>: É um microframework para desenvolvimento web escrito em Python. É conhecido por ser leve, flexível e minimalista, fornecendo as ferramentas essenciais para construir aplicações web e APIs de forma rápida.
  </li>
  <li>
    <a href="https://git-scm.com/">Git</a>: É um sistema de controle de versão distribuído, gratuito e de código aberto. Ele permite que desenvolvedores rastreiem mudanças no código-fonte, gerenciem o histórico de alterações e colaborem em projetos de software simultaneamente.
  </li>
  <li>
    <a href="https://getbootstrap.com/">Bootstrap</a>: É um framework front-end de código aberto focado no desenvolvimento de sites responsivos e voltados para dispositivos móveis (mobile-first). Oferece uma vasta biblioteca de componentes prontos em HTML, CSS e JavaScript.
  </li>
</ul>


<h2 style="font-family:roboto;"> Contribuições Individuais :dart:</h2>
  
  <h3> Atribuições como Desenvolvedor Back-end e Front-end</h3>
  <p align="justify" style="font-family:roboto;"> Como desenvolvedor nesse projeto, minhas atribuições principais envolveram a estruturação e estilização das interfaces de usuário para garantir uma navegação intuitiva, além da construção da lógica de back-end responsável por avaliar o aprendizado dos usuários sobre os conceitos ágeis.


Minha contribuição no Front-end, utilizando HTML e CSS, permitiu a criação de páginas semânticas e layouts responsivos baseados no design de referência. Foquei no uso de Flexbox para alinhamento e centralização, garantindo que a experiência visual fosse agradável e consistente em qualquer dispositivo durante a resolução das atividades.
